# Trabajo Practico - Grupo 1 - Ingenieria de la Rehabilitacion 

### 🎲 Box & Blocks Test (BBT)
Este test consiste en dos pruebas de un minuto en las que el usuario utilizará una mano para controlar el mouse y mover los bloques. La idea es mover la mayor cantidad de bloques de una caja a la otra sin tocar la división (línea central entre las cajas). Consta de dos pruebas, una con los bloques del lado derecho y otra con los bloques del lado izquierdo. 

Se miden los éxitos (bloques arrastrados hacia la otra caja), las veces que un bloque se soltó, las veces que se tocó la pared inicial, velocidad del mouse (promedio y máxima), picos de aceleracion del mouse, espasmos, cambios de direccion bruscos, entre otros.
