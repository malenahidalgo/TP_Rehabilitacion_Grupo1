# Trabajo Practico - Grupo 1 - Ingenieria de la Rehabilitación 
Bienvenidx al repositorio del Trabajo Práctico del Grupo 1 de la materia Ingeniería de la Rehabilitación. En este repositorio encontrará 3 tests con sus respectivas carpetas, es necesario que descargue las carpetas completas para poder probarlos. 

1. Descargar carpeta completa
2. Descomprimirla
3. Ejecutar "pip install requirements" en la terminal (mejor explicado en los readme.md de las carpetas)
4. Ejecutar el main.py

### 👥 Población dirigida
El test está diseñado para pacientes en proceso de **rehabilitación neurocognitiva y motriz**, particularmente aquellos con secuelas de **ACV (Accidente Cerebrovascular)** que presenten hemiparesia.
