# Trabajo Practico - Grupo 1 - Ingenieria de la Rehabilitación 

## 🧠 Test de Reacción Multimodal 
### 📊 Qué mide el test
Este software realiza una **Evaluación de Reacción Multimodal y Control Motor**. Específicamente mide:
* Tiempos de reacción ante estímulos visuales y auditivos (Interferencia).
* Estabilidad atencional y fatiga cognitiva.
* Precisión motriz mediante la **Ley de Fitts**, analizando la rectitud y fluidez de la trayectoria del cursor hacia el objetivo.

### 👥 Población dirigida
El test está diseñado para pacientes en proceso de **rehabilitación neurocognitiva y motriz**, particularmente aquellos con secuelas de **ACV (Accidente Cerebrovascular)** que presenten dificultades en la coordinación visomotora o heminegligencia (evaluada mediante el análisis de lateralidad por cuadrantes).
